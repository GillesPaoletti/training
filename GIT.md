GIT LINKS

Resources to learn Git: https://try.github.io/ 
Check the English cheatsheet at https://github.github.com/training-kit/downloads/github-git-cheat-sheet.pdf

Git itself can be downloaded from https://git-scm.com/

The entire Pro Git book, written by Scott Chacon and Ben Straub and published by Apress, is available at https://git-scm.com/book/en/v2.

Git Immersion ( http://jce-il.github.io/git-immersion/index.html ) is a guided tour that walks Git users through the fundamentals of Git.
